# 🎓 EduPredict: Student Performance Prediction & Intervention Recommendation System
EduPredict is an end-to-end Machine Learning and Explainable AI solution designed to identify students at risk of academic failure or dropout and provide actionable intervention strategies for educators.

The project leverages the Open University Learning Analytics Dataset (OULAD) and applies advanced feature engineering, predictive modeling, and explainable AI techniques to build an intelligent early warning system.

🚀 Key Features
Student Performance Prediction using:
Logistic Regression
Random Forest
XGBoost
Advanced Feature Engineering
Engagement metrics
Learning behavior indicators
Assessment performance features
Registration and completion analytics
Explainable AI (XAI)
SHAP (SHapley Additive Explanations)
Feature importance analysis
Personalized risk-factor identification
Recommendation Engine
Detects key reasons behind student risk
Generates personalized educator interventions
Supports mentoring, tutoring, counseling, and engagement strategies
Interactive Streamlit Dashboard
Real-time predictions
Risk visualization
Student performance analytics
Educator recommendation panel

🎯 Project Outcome

The system not only predicts whether a student is at risk but also explains why the prediction was made and recommends personalized actions educators can take to improve student outcomes and reduce dropout rates.
